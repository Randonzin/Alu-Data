O AluData é um sistema de gerenciamento escolar desenvolvido como projeto acadêmico no curso de Análise e Desenvolvimento de Sistemas. O objetivo do projeto é aplicar, de forma prática, os conceitos de CRUD (Create, Read, Update e Delete), integrando backend, frontend e banco de dados.

A aplicação utiliza Python com Flask para criação da API, SQLite para persistência dos dados e uma interface web construída com HTML, CSS e JavaScript, permitindo o cadastro, visualização, edição e exclusão de alunos de forma simples e funcional.

O projeto simula uma aplicação real de controle escolar, servindo como base para aprendizado em desenvolvimento web e integração de sistemas.
